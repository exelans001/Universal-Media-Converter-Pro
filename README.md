# Universal Media Converter Pro 🎬⬇️

A modern, fast, and open-source media downloader and local file converter built with Python and CustomTkinter. 

Whether you want to download videos from platforms like YouTube or convert local media files to different formats, this tool handles it with a sleek, user-friendly, and highly optimized UI.

## ✨ Features
* **URL Downloader:** Download videos/audio directly from supported websites via `yt-dlp`.
* **Local Converter:** Convert existing local media files into various formats (MP4, MKV, AVI, MP3, WAV, etc.) using `FFmpeg`.
* **Optimized Performance:** Uses multithreading and UI update throttling to prevent freezing and stuttering during high-speed downloads.
* **Bilingual Support:** Fully supports both English and Turkish interfaces out of the box.
* **Modern UI:** Built with CustomTkinter for a beautiful Dark/Light mode interface.

## 🚀 How to Use (For Regular Users)
If you just want to use the program without dealing with code, you can download the ready-to-use `.exe` version.

1. Go to the **[Releases](../../releases)** page.
2. Download the latest `Universal_Media_Converter_Pro.zip` file.
3. Extract the ZIP file to a folder.
4. Run the `.exe` file and enjoy!

*(Note: The `ffmpeg` binaries are handled automatically via `imageio-ffmpeg`, so no extra installation is required).*
